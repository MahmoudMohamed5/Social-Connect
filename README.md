# Social Connect - Bootstrap 4 Landing Page Template

A clean, modern, responsive landing page template tailored for SaaS platforms, social media apps, or digital marketing tools. Built using **Bootstrap 4**, **FontAwesome**, and structured HTML5/CSS3.

## 🚀 Demo & Key Features

* **Responsive Navigation:** A fixed dark-themed navbar utilizing Bootstrap's responsive collapse system.
* **High-Converting Hero Section:** A striking background header split between feature benefits (with structured d-flex checkmark lists) and an embedded registration card.
* **Feature Sections ("Explore" & "Create"):** Alternating image/text utility grids with rounded image geometry (`img-fluid rounded-circle`).
* **Fully Functional Modal:** Built-in "Contact Us" trigger that seamlessly loads an overlay contact form.
* **Optimized Performance:** Uses the slim version of jQuery alongside Bootstrap Bundle for minimal overhead.

---

## 🛠️ Tech Stack & Dependencies

* **HTML5 & CSS3** (Custom style architecture in `css/style.css`)
* **Bootstrap 4.x** (Layout engine & components)
* **FontAwesome 4/5** (Icon vector engine)
* **jQuery Slim** (DOM manipulation for Bootstrap animations)
